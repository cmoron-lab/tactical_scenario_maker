🤖 Générateur de scénarios IA

Décrivez votre scénario tactique en langage naturel. L'IA générera automatiquement les agents, les tâches, les méthodes HTN et les conditions. Les tâches manquantes seront ajoutées à la base de connaissances.

💡 Comment bien écrire votre description

Comportement conditionnel — un agent peut réagir différemment selon une variable, avec une ligne par cas :
si commande = cercle : va vers agent 1 et encercle-le si commande = veille : ne rien faire

Comportement selon la distance — reconnu directement, ça correspond à ce que "éviter"/"encercler" savent déjà faire (approche au loin, réaction une fois proche) :
si distance(agent1) > 100 m : rejoindre l'agent 1 si distance(agent1) ≤ 100 m : tourner autour de l'agent 1
