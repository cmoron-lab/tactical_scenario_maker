# Éditeur v2 de scénarios (Scenario Request) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rendre les scénarios v2 (forces, relations, zones, agents+missions, triggers, fin de scénario) créables et éditables dans l'IHM, avec sauvegarde validée côté serveur et validation à l'édition contre un profil d'exécution (§4.5 de docs/lsga-architecture-v3.md : l'incompatibilité se voit AVANT le lancement).

**Architecture :** Côté serveur, `Api.save_scenario` branche sur `version == 2` (validation par `ReferenceScenario.from_dict`, écriture par un nouveau `save_reference_scenario`), et un endpoint `POST /api/scenario/<name>/validate` assemble `from_dict` + `required_capabilities_for_task` + `validate_profile` (aucun ROS). Côté IHM, un objet JS `v2doc` est LA source de vérité (chargé au select, muté par les handlers, POSTé au save) — jamais de reconstruction depuis le DOM, c'est le piège n°1 du v1 (`collectAgents` perd les champs non rendus). `renderEditorV2` remplace `renderReadOnlyScenario` ; chaque carte (Zones, Forces, Relations, Agents, Triggers, Fin) a son `renderV2X()` qui réécrit son conteneur depuis `v2doc`.

**Tech Stack :** Python stdlib (http.server), JS vanilla inline dans `templates/index.html` (conventions existantes : `build*`/`render*`/`update*`, classes CSS `agent-card`/`exec-info-row`/`field`), pytest.

## Global Constraints

- Python ≥ 3.10, stdlib uniquement côté runtime ; messages d'erreur en FRANÇAIS.
- Aucun import ROS hors `tsm/lotusim/client.py` et `tsm/execution/runtime.py` (`tsm.execution.controller` est importable côté serveur — docstring « Aucun import ROS ici »).
- JS : vanilla inline, portée globale (handlers `onclick`/`onchange` inline), pas de framework, pas de module. Respecter les préfixes existants (`render*` réécrit un bloc via innerHTML, `build*` construit du HTML pur, `update*` rafraîchit un sous-widget).
- Les noms (scénarios, zones, forces, agents) sont contraints par `_NAME_RE` côté serveur — l'interpolation directe dans les handlers inline suit l'idiome v1 (`onclick="selectScenario('${n}')"`).
- Les 3 seuls types de condition du langage : `in_zone` (agent, zone), `all_in_zone` (force, zone), `agent_destroyed` (agent OU force). La seule action de trigger : `spawn_force` (force).
- La seule mutation de `templates/index.html` : ajouter/remplacer les fonctions nommées dans les tâches — ne pas toucher à l'éditeur v1 ni aux onglets KB/Exécution.
- `uv run pytest`, `uv run ruff check .`, `uv run mypy` verts après chaque tâche.
- Un scénario v2 renommé reste contraint par le profil : les CLÉS d'agents du profil doivent couvrir les agents du scénario (l'éditeur de profils est un incrément futur, hors périmètre).

```python
def v3_mission_tasks(kb: Mapping[str, Any]) -> list[str]:
    """Tâches offrables comme mission d'un scénario v2 : les tâches Python
    (décomposition figée dans _PYTHON_TASK_CAPABILITIES) et les tâches

    return sorted(names)
```

```python
def validate_scenario(self, doc: Any, profile_name: Any) -> dict[str, Any]:
    """Validation à l'édition (§4.5) : schéma v2 + couple scénario/profil.
    Le résultat est la donnée (toujours 200, y compris profil inconnu —
    ProfileError est un résultat) ; seul un corps incomplet fait un 400."""
    if not isinstance(doc, dict) or not isinstance(profile_name, str) or not profile_name:
        raise ScenarioError("corps attendu: {'scenario': <document v2>, 'profile': '<nom>'}")
    try:
        scenario = ReferenceScenario.from_dict(doc)
        profile = load_profile(profile_name)
        kb = doctrine.load()
        required = {agent: required_capabilities_for_task(kb, spec.mission.task)
                    for agent, spec in scenario.agents.items()}
        validate_profile(scenario, profile, required)
    except (ScenarioError, ProfileError) as e:
        return {'ok': False, 'errors': [str(e)]}
    return {'ok': True, 'errors': []}
```

```js
function renderV2Zones() {
  const rows = Object.entries(v2doc.zones).map(([zname, z]) => `
    <div class="form-grid">
      <div class="field"><label>Nom</label>
        <input value="${escHtml(zname)}" onchange="renameV2Zone('${zname}', this.value)"></div>
      <div class="field"><label>Latitude</label>
        <input type="number" step="any" value="${z.center.lat}"
          onchange="v2doc.zones['${zname}'].center.lat = parseFloat(this.value)"></div>
      <div class="field"><label>Longitude</label>
        <input type="number" step="any" value="${z.center.lon}"
          onchange="v2doc.zones['${zname}'].center.lon = parseFloat(this.value)"></div>
      <div class="field"><label>Rayon (°)</label>
        <input type="number" step="any" value="${z.radius_deg}"
          onchange="v2doc.zones['${zname}'].radius_deg = parseFloat(this.value)"></div>
      <div class="field"><label>&nbsp;</label>
        <button class="agent-remove" onclick="deleteV2Zone('${zname}')">Supprimer</button></div>
    </div>`).join('');
  document.getElementById('v2-zones').innerHTML = `
    <div class="agent-card">
      <div class="agent-head"><span class="agent-name-display">Zones</span>
        <button class="btn btn-ghost" onclick="addV2Zone()">+ Zone</button></div>
      <div class="exec-card-body">${rows || '<p class="exec-empty">Aucune zone.</p>'}</div>
    </div>`;
}

function addV2Zone() {
  let n = 1;
  while (v2doc.zones[`zone_${n}`]) n++;
  v2doc.zones[`zone_${n}`] = { center: { lat: 26.5520, lon: 56.4000 }, radius_deg: 0.0002 };
  renderV2Zones();
}

function renameV2Zone(oldName, newName) {
  newName = newName.trim();
  if (!newName || newName === oldName || v2doc.zones[newName]) { renderV2Zones(); return; }
  v2doc.zones[newName] = v2doc.zones[oldName];
  delete v2doc.zones[oldName];
  renderV2Zones();
}

function deleteV2Zone(name) {
  delete v2doc.zones[name];
  renderV2Zones();
}

function showV2Status(msg, isError) {
  document.getElementById('launch-status').innerHTML =
    `<div class="exec-info-row"><span class="exec-info-val" style="color:${isError ? 'var(--red, #b3261e)' : 'inherit'}">${escHtml(msg)}</span></div>`;
}

function saveScenarioV2() {
  const name = document.getElementById('sc-name-inp').value.trim();
  if (!name) { showV2Status('Nom de scénario requis.', true); return; }
  fetch(`/api/scenario/${name}`, {
    method: 'POST', headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(v2doc),
  }).then(r => r.json().then(d => ({ httpOk: r.ok, d })))
    .then(({ httpOk, d }) => {
      if (!httpOk) { showV2Status(d.error || 'Erreur de sauvegarde', true); return; }
      current = name;
      document.getElementById('sc-name-inp').setAttribute('readonly', '');
      showV2Status('Scénario enregistré.', false);
      loadList();
    });
}

function validateScenarioV2() {
  const profile = document.getElementById('profile-select')?.value;
  if (!profile) { showV2Status('Choisir un profil pour valider.', true); return; }
  fetch(`/api/scenario/${current || 'nouveau'}/validate`, {
    method: 'POST', headers: { 'Content-Type': 'application/json' },
```

```js
function renameV2Force(oldName, newName) {
  newName = newName.trim();
  if (!newName || newName === oldName || v2doc.forces[newName]) { renderV2Forces(); renderV2Relations(); return; }
  v2doc.forces[newName] = v2doc.forces[oldName];
  delete v2doc.forces[oldName];
  // répercuter dans les relations (from/to) — sinon référence pendante
  for (const r of v2doc.relations) {
    if (r.from === oldName) r.from = newName;
    r.to = r.to.map(t => t === oldName ? newName : t);

}

function deleteV2Force(name) {
```

```js
const V2_CONDITION_TYPES = [
  { id: 'in_zone', label: 'agent dans une zone' },
  { id: 'all_in_zone', label: 'toute une force dans une zone' },
  { id: 'agent_destroyed', label: 'destruction (agent ou force)' },
];

function v2CondAt(loc, i) {
  return loc === 'trigger' ? v2doc.triggers[i].when : v2doc.end[loc][i];
}

function setV2CondType(loc, i, type) {
  const fresh = {
    in_zone: { type, agent: '', zone: '' },
    all_in_zone: { type, force: '', zone: '' },
    agent_destroyed: { type, agent: '' },
  }[type];
  if (loc === 'trigger') v2doc.triggers[i].when = fresh;
  else v2doc.end[loc][i] = fresh;
  renderV2Triggers(); renderV2End();
}

function setV2CondField(loc, i, field, value) {
  const c = v2CondAt(loc, i);
  if (field === 'porteur') {           // agent_destroyed : agent OU force
    delete c.agent; delete c.force;
    const [kind, name] = value.split(':');
    if (name) c[kind] = name;
    return;
  }
  c[field] = value;
}

function buildV2ConditionForm(loc, i) {
  const c = v2CondAt(loc, i);
  const zoneSel = (field) => `
    <div class="field"><label>Zone</label>

      </select></div>`;
  let fields = '';
  if (c.type === 'in_zone') {
    fields = `
      <div class="field"><label>Agent</label>
        <select onchange="setV2CondField('${loc}', ${i}, 'agent', this.value)">
          <option value="">—</option>
          ${Object.keys(v2doc.agents).map(a => `<option value="${a}" ${c.agent === a ? 'selected' : ''}>${escHtml(a)}</option>`).join('')}

  } else if (c.type === 'all_in_zone') {
    fields = `
      <div class="field"><label>Force</label>
```
